# EventPoster-Mobile

## Описание приложения
### Назначение
Мобильное приложение по поиску подходящих событий упрощает процесс поиска и покупки билетов на мероприятия. Оно предоставляет пользователям возможность быстро и удобно оформлять покупку билетов, просматривать события.
### Цели
*	Упростить процесс поиска интересующих событий.
*	Обеспечить пользователям удобный и простой в использовании способ покупки билетов.
### Перечень задач на реализацию:
*	Регистрация и вход. Создание аккаунта, вход через социальные сети, авторизация.
*	Просмотр каталога событий. Просмотр каталога с фотографиями событий, описанием, ценами.
*	Избранное. Добавление и сохранение в Избранное интересующих пользователя событий.
*	Оформление покупки билета. Выбор события, выбор посадочных мест, выбор и добавление способа оплаты.
*	Отслеживание билетов. Сохранение и отображение информации о приобретенных билетах.

## Описание инфраструктуры
База данных: MS SQL Server

Сервер: ASP.NET
